# Pokymon

Pokymon is a Pokémon-like RPG videogame developed while taking [this Unity course on Udemy](https://www.udemy.com/share/106hyQ3@xr27n-py3L5eQSWov5eMF2nOb4sNUg72v-tENWzunyGr3MIeZa0yzkEfk6i04t01/).
